    Importação em Lotes

    A importação em lotes tem como objetivo facilitar e agilizar o processo de configuração do painel. Os dados são inseridos através de arquivos no formato CSV gerados pelo Excel. Os dados são separados por colunas, e cada tipo de importação segue uma ordem que os dados são cadastrados.
